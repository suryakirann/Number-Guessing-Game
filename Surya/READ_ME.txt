#################################

   Playing Number Guessing Game

#################################

Operating Syatem: Windows

Instructions for Playing the game:
- Download the NumberGuessingGame.exe file
- Double Click for playing the game

Note:
You cannot choose a different number once the game starts.
If you do, the game ends with "illegal Number of attempts"

Files Attached:
- java class file (GuessingGamenumber.class)
- Config file
- NumberGuessingGame.jar file
- NumberGuessingGame.exe file
